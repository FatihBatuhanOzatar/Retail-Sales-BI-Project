# Dashboards and Data Visualization

This directory contains the visual layer of the Retail Sales Analysis project.

## Components
* **Power_BI_project.pbix:** The master file containing the data model, DAX measures, and interactive reports.
* **/visuals:** High-resolution screenshots of the primary dashboard pages for quick review.

## Analysis Focus
1. **Executive Summary:** High-level overview of Sales, Profit, and Shipping performance.
2. **Product Profitability:** Detailed breakdown of category-level margins and discount correlations.
3. **Operational Efficiency:** Analysis of shipping lag and return indicators across different segments.

## Technical Implementation
* **DAX Measures:** Custom calculations for Profit Margin, Return Rates, and Year-over-Year growth.
* **Star Schema:** Optimized data modeling to ensure high performance and intuitive filtering.
* **Interactivity:** Dynamic slicers for Region, Category, and Time-series analysis.
