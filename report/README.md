# Technical and Strategic Reports

This directory houses the documentation regarding the methodology, analytical findings, and business recommendations.

## Featured Document
* **BI_Project_Retail_Sales_Report.pdf:** The final project report submitted for the Business Intelligence and Reporting Tools course at Istanbul University.

## Report Content
* **Data Cleaning:** Detailed account of handling outliers and missing values using Python.

The analysis follows a data-driven approach, converting raw transactional records into strategic business intelligence through rigorous cleaning, modeling, and visualization phases.
